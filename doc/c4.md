# C4 : Presentation

Este capítulo apresenta os elementos associados à demonstração final do projeto **Car4Me – API de Gestão de Aluguer de Veículos**.

A apresentação foi desenvolvida de forma clara, objetiva e visual para comunicar:

- a arquitetura da solução,  
- o modelo de dados implementado,  
- o funcionamento dos principais endpoints.
- implementação do docker

## 4.3 Utilização de Inteligência Artificial no Projeto

- A inteligência artificial foi utilizada como ferramenta de **apoio ao desenvolvimento**.  
- Auxiliou na **escrita e refatoração** do ficheiro OpenAPI (YAML), nomeadamente na definição de endpoints, parâmetros e esquemas de dados.  
- Serviu de apoio na **definição e validação da lógica** de alguns endpoints da API (Node.js), incluindo tratamento de erros e códigos de resposta HTTP.  
- Foi usada para **esclarecer e melhorar a lógica** de triggers na base de dados, garantindo a consistência entre veículos, reservas e manutenções.  
- Contribuiu para a **organização e revisão do texto** do relatório, sugerindo melhorias de estrutura, clareza e correção linguística.  
- Todas as sugestões geradas pela inteligência artificial foram **analisadas, adaptadas e validadas manualmente**, assegurando que as decisões finais refletem o entendimento e o trabalho autónomo dos autores.
